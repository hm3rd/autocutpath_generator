\# Face-Centered Rotating Overlay (Python)



Webカメラの顔に画像を重ね、顔の傾き（ロール角）に合わせて画像も回転させて表示します。MediaPipe FaceMesh と OpenCV を使用しています。



\## セットアップ



1\. 依存ライブラリをインストール（仮想環境がある場合は有効化してから）



```powershell

pip install -r requirements.txt

```



> VS Code 内部の仮想環境を使用している場合は、ターミナルがその環境を指していることを確認してください。



2\. 重ねたい画像ファイルを `overlay.png` という名前で、このフォルダ直下に置きます（透過PNG推奨）。

&#x20;  - 画像が見つからない場合は、サンプルの赤い円スタンプが自動生成されます。



\## 実行



```powershell

python test.py

```



\- ウィンドウにカメラ映像が表示され、顔の中心に画像が重なります。

\- 顔を傾けると、画像も同じ角度に回転します。

\- 終了するには `q` を押します。



\## 調整ポイント

\- 画像サイズ: 顔幅（目の間隔）×1.2 を基準に自動調整しています。倍率は `test.py` 内の `face\_w \* 1.2` を変更してください。

\- 回転の基準: 左右の目の外端ランドマーク（33, 263）からロール角を推定しています。

\- 表示位置: 目の中点を顔の中心とみなしています（必要なら鼻先などに変更可能）。
