# autocutpath_generator
アクリルスタンドなどのグッズを作成する際のベクターファイル(.svg)を自動で作成するアプリです。

## インストールに関して
### 実行ファイル(.exe)から実行する場合
autocutpath_generator.zipを解凍し、実行する。windowsのセキュリティに引っ掛かるが実行できる。

> [!WARNING]
もちろん悪意のあるファイルではないが、何かあった場合は自己責任でお願いします。

###プログラムから実行する場合
requirements.txtからライブラリをインストールする

## 実行
1. 実行ファイルを実行(もしくはプログラムから実行)するとGUIが立ち上がる

2. 作成したい画像(.png)を指定する。
> [!NOTE]
ここで使用する画像は作成したい部分が切り取られている(透過ファイル)前提です。

> 左側に入力画像が表示されます。

3. 出力したいフォルダを選択する

4. プレビューボタンを押すと右側にプレビューが表示されます。

> パラメータを変更した際は、再度プレビューボタンを押すと更新されます。
5. SVG作成ボタンを押すと、指定したフォルダにファイルが保存されます。
